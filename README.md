# Project Setup
## Prerequisites
Node.js (v18 or higher)
npm
## Installation
From the project root folder `fundamentals-project-template/`, run:

npm install
## Run the project
From the same root folder fundamentals-project-template/, run:

npm run dev
## Note
All commands should be executed from the root directory fundamentals-project-template/.

## This command npm run dev:

Automatically compiles all .scss files from src/scss/ to dist/css/ 

Starts the project and opens it in the browser via live-server

Runs ESLint and Stylelint before launching to ensure code quality
